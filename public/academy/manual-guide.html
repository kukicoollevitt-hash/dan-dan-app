1 학생 계정 생성 및 관리 ▼

📌 학생 등록 방법

  • 경로: 대시보드 → 내 학원 학생 목록 → [학생 추가] 버튼
  • 필수 입력: 학년, 이름, 초기 비밀번호
  • 학년 형식: "초3", "초4", "초5", "초6", "중1", "중2", "중3" 등

📌 학생 정보 수정

  • 학생 목록에서 해당 학생의 [수정] 버튼 클릭
  • 학년, 이름, 비밀번호 변경 가능

📌 비밀번호 초기화

  • 학생이 비밀번호를 잊어버린 경우 선생님이 직접 초기화
  • 초기화 후 학생에게 새 비밀번호 안내 필요
💡 TIP: 초기 비밀번호는 학생이 기억하기 쉬운 것으로 설정하고, 첫 로그인 후 변경하도록 안내하세요.

3 단원 학습 5단계 Flow ▼

🔄 학습 진행 순서

단계 활동 설명
1단계 본문 읽기 지문을 읽으며 각 문장 클릭 → 완독 시 축하 효과
2단계 중심내용 찾기 각 문단의 핵심 문장 선택 (보통 4~5개 문단)
3단계 문제 풀이 5가지 문해력 영역별 문제 (각 1문제씩, 총 5문제)
4단계 창의활동 배운 내용 바탕으로 자유 글쓰기
5단계 분석리포트 레이더 차트로 5가지 영역 점수 확인
💡 수업 진행 TIP: 본문 읽기 → 중심내용 → 문제 순서를 꼭 지키도록 안내하세요. 순서대로 해야 정확한 분석이 가능합니다.
4 5가지 문해력 지수 이해 ▼

📊 문해력 5대 지수

지수 측정 내용 문제 유형 예시
1. 사실적 이해 글에 명시된 정보를 정확히 파악하는 능력 "본문에 따르면 ~은 무엇인가요?"
2. 구조적 이해 글의 구조와 논리적 흐름을 파악하는 능력 "이 글의 전개 방식은?", "문단의 역할은?"
3. 어휘력 핵심 어휘의 의미를 문맥에서 파악하는 능력 "밑줄 친 단어의 의미와 같은 것은?"
4. 추론적 이해 글에 드러나지 않은 의미를 유추하는 능력 "글쓴이가 말하고자 하는 바는?"
5. 비판적 이해 글의 내용을 평가하고 판단하는 능력 "이 주장의 한계점은?", "반박할 수 있는 근거는?"
ℹ️ 레이더 차트 해석: 5가지 영역 중 안쪽으로 들어간 부분이 학생의 취약 영역입니다. 해당 영역 보완 학습을 권장하세요.
5 형성평가(관문) 시스템 ▼

🚪 관문 시스템 개요

  • 도전 조건: 5개 단원 학습 완료 시 관문 도전 가능
  • 문제 구성: 총 10문제 (학습한 5개 단원에서 각 2문제씩 출제)
  • 통과 기준: 10문제 중 8문제 이상 정답 (80% 이상)
  • 재도전: 통과 실패 시 무제한 재도전 가능

🎯 관문별 범위

관문 학습 범위
관문 101~05 단원
관문 206~10 단원
관문 311~15 단원
관문 416~20 단원
⚠️ 주의: 관문을 통과해야 다음 5개 단원이 해금됩니다. 학생이 진도가 막혀있다면 관문 통과 여부를 확인하세요.
6 학습 현황 모니터링 ▼

📈 전체 현황 확인

  • 경로: 대시보드 → 내 학원 학생 목록
  • 학생별 현재 학습 진행률 확인
  • 최근 접속일 및 학습 활동 확인

👤 개별 학생 상세 기록

  • 학생 이름 클릭 → 상세 학습 기록 페이지 이동
  • 단원별 완료 여부, 점수, 소요 시간 확인
  • 5가지 문해력 지수별 평균 점수 확인

🏆 관문 통과 기록

  • 경로: 대시보드 → 행동예측 데이터 목록
  • 관문별 통과 날짜, 재도전 횟수 확인
  • 관문 뱃지 클릭 시 상세 정보 팝업
7 행동예측 AI 데이터 활용 ▼

🤖 수집되는 행동 데이터

데이터 항목 의미 활용법
본문 읽기 시간 지문을 읽는 데 걸린 총 시간 너무 빠르면 대충 읽은 것, 너무 느리면 이해 어려움
중심문단 오답 중심내용 찾기에서 틀린 횟수 구조 파악 능력 취약 여부 판단
문제 오답 5가지 영역별 문제 오답 횟수 취약 영역 파악 (사실/구조/어휘/추론/비판)
관문 재도전 횟수 형성평가 통과까지 시도한 횟수 전반적 학습 완성도 판단
💡 상담 활용: 행동 데이터를 학부모 상담 시 객관적 근거 자료로 활용하세요.
8 진단테스트 운영 ▼

📝 진단테스트란?

  • 학생의 현재 문해력 수준을 진단하는 테스트
  • 5가지 문해력 영역별 점수 산출
  • 맞춤 학습 커리큘럼 추천에 활용

📋 운영 방법

  • 신청 확인: 대시보드 → 진단테스트 신청 목록
  • 테스트 진행: [진단테스트 바로가기] 버튼으로 학생 안내
  • 결과 확인: 테스트 완료 후 자동으로 결과 리포트 생성
ℹ️ 진단테스트는 학원 등록 전 또는 학기 초에 실시하면 효과적입니다.
9 학부모 상담 자료 활용 ▼

📊 활용 가능한 자료

  • 학습 진행률: 전체 단원 중 완료한 비율
  • 레이더 차트: 5가지 문해력 영역별 점수 시각화
  • 취약 영역 분석: 가장 낮은 점수 영역 및 보완 방법
  • 성장 추이: 시간에 따른 점수 변화 그래프
  • 관문 통과 기록: 재도전 횟수, 통과 날짜

💬 상담 포인트

  • "OO 학생은 추론적 이해 영역이 취약합니다. 글의 숨은 의미를 파악하는 연습이 필요해요."
  • "관문 통과까지 평균 1.5회 시도했습니다. 꾸준히 학습하고 있어요."
  • "지난 달 대비 어휘력 점수가 15% 상승했습니다."
10 자주 묻는 질문 (FAQ) ▼

❓ 학생이 로그인이 안 된다고 해요

  • 학년/이름/비밀번호 정확히 입력했는지 확인
  • 학년 형식 확인 (예: "초4" ← O, "4학년" ← X)
  • 비밀번호 초기화 후 재시도

❓ 학생의 학습 데이터를 초기화하고 싶어요

  • 개별 단원 초기화는 불가능합니다.
  • 필요시 본사에 문의해 주세요.

❓ 관문을 통과했는데 다음 단원이 안 열려요

  • 페이지 새로고침(F5) 시도
  • 로그아웃 후 재로그인
  • 캐시 삭제 후 재접속

❓ 창의활동 글이 저장이 안 돼요

  • 인터넷 연결 상태 확인
  • 글자 수 제한(최소 50자) 확인
  • [제출] 버튼을 꼭 클릭했는지 확인
1 로그인 방법 ▼

🔐 로그인하기

  • 접속 주소: 브레인문해력 홈페이지 또는 선생님이 안내한 주소
  • 입력 정보: 학년 + 이름 + 비밀번호
  • 학년 입력 예시: "초3", "초4", "중1" 등

🔑 비밀번호를 잊어버렸어요

  • 선생님께 말씀드리면 비밀번호를 초기화해 주실 거예요.
  • 새 비밀번호를 받으면 로그인 후 꼭 변경하세요!
💡 같은 기기에서는 자동 로그인이 되어 편리해요!

3 본문 학습하기 ▼

📖 지문 읽기

  • 글을 처음부터 끝까지 천천히 읽어요.
  • 읽은 문장을 클릭하면 하이라이트가 켜져요.
  • 모든 문장을 클릭하면 "완독 축하!" 메시지가 나와요! 🎉

💡 잘 읽는 방법

  • 모르는 단어가 나오면 문맥을 보고 뜻을 추측해 봐요.
  • 중요한 부분은 여러 번 읽어도 괜찮아요.
  • 급하게 읽지 말고 내용을 이해하면서 읽어요.
ℹ️ 읽기 시간도 기록되니까 너무 빨리 넘기지 마세요!
4 중심내용 찾기 ▼

🎯 중심내용이란?

  • 각 문단에서 가장 중요한 내용을 담은 문장이에요.
  • 보통 문단의 처음이나 끝에 있는 경우가 많아요.

✅ 찾는 방법

  • 문단을 읽고 "이 문단이 하고 싶은 말이 뭘까?" 생각해요.
  • 여러 보기 중에서 가장 핵심적인 내용을 선택해요.
  • 틀려도 괜찮아요! 다시 선택할 수 있어요.
5 문제 풀기 ▼

📝 5가지 문제 유형

문제 무엇을 물어보나요?
1번 글에 직접 나온 내용 (사실적 이해)
2번 글의 구조와 흐름 (구조적 이해)
3번 단어의 뜻 (어휘력)
4번 글에서 추측할 수 있는 것 (추론적 이해)
5번 글에 대한 나의 생각 (비판적 이해)

💡 문제 풀이 팁

  • 문제를 읽고 본문에서 관련 부분을 다시 찾아봐요.
  • 어려우면 힌트를 확인해 봐요.
  • 틀려도 다시 풀 수 있으니 걱정 마세요!
6 어휘학습 ▼

📚 어휘학습이란?

  • 본문에 나온 핵심 어휘 5개를 배워요.
  • 빈칸에 알맞은 단어를 넣는 문제예요.

✏️ 학습 방법

  • 뜻을 보고 알맞은 단어를 입력해요.
  • 모르겠으면 힌트(첫 글자)를 확인해요.
  • 다 풀면 [채점하기] 버튼을 눌러요!
💡 어휘를 많이 알면 글을 더 쉽게 이해할 수 있어요!
7 창의활동 작성하기 ▼

✏️ 창의활동이란?

  • 배운 내용을 바탕으로 나만의 글을 쓰는 활동이에요.
  • 정답이 없어요! 자유롭게 생각을 표현하면 돼요.

📝 작성 방법

  • 글다듬기: 주어진 문장을 더 좋게 수정해요.
  • 창의적 글쓰기: 주제에 맞게 자유롭게 써요.
  • 막막하면 힌트를 참고해요.
  • 최소 50자 이상 써야 제출할 수 있어요.
  • 다 쓰면 [제출] 버튼을 꼭 눌러요!
⚠️ 제출 버튼을 누르지 않으면 저장이 안 돼요!
8 분석리포트 보기 ▼

📊 분석리포트란?

  • 단원 학습 후 [분석리포트] 탭을 클릭해요.
  • 레이더 차트로 나의 실력을 한눈에 볼 수 있어요.
  • 5가지 영역 중 안쪽으로 들어간 부분이 부족한 부분이에요.

📈 성장 확인하기

  • 시간이 지나면서 레이더 차트가 점점 커지는 걸 확인해요.
  • 부족했던 영역이 나아지는 것을 눈으로 볼 수 있어요!
  • AI가 학습 권장 사항을 알려줘요.
ℹ️ 꾸준히 학습하면 실력이 쑥쑥 늘어요! 화이팅! 🎉

  • 예전에 배운 단원을 다시 학습할 수 있어요.
  • 완료한 단원 수가 늘어나는 것을 확인해요.
  • 어떤 과목을 많이 배웠는지 한눈에 볼 수 있어요.
ℹ️ 고래 도서관이 책으로 가득 차면 뿌듯해요! 📚
11 🏅 고래 배지 ▼

🎖️ 고래 배지란?

  • 학습 성취를 달성하면 받는 특별한 배지예요.
  • 다양한 종류의 배지를 모을 수 있어요!

🏆 배지 종류 예시

  • 첫 학습 배지: 첫 단원 완료 시
  • 완독왕 배지: 본문을 꼼꼼히 읽었을 때
  • 관문 통과 배지: 관문을 통과할 때마다
  • 연속 학습 배지: 며칠 연속 학습 시
  • 어휘왕 배지: 어휘 문제 많이 맞출 때
💡 배지를 많이 모으면 친구들에게 자랑할 수 있어요! ✨
12 🤖 AI 고래쌤 ▼

🐋 AI 고래쌤이란?

  • 학습 중 궁금한 것을 물어볼 수 있는 AI 튜터예요.
  • 언제든지 대화하며 도움을 받을 수 있어요.

💬 이렇게 활용해요

  • 개념 설명 요청: "이 단어가 무슨 뜻이야?"
  • 문제 힌트: "이 문제 어떻게 풀어?"
  • 추가 정보: "이것에 대해 더 알려줘"
  • 학습 조언: "어떤 부분을 더 공부해야 해?"
ℹ️ AI 고래쌤은 친절하게 설명해 줘요. 부끄러워하지 말고 물어보세요! 🐳
13 📝 어휘퀴즈 ▼

🎯 어휘퀴즈란?

  • 학습한 어휘를 복습하는 퀴즈예요.
  • 랜덤으로 어휘 문제가 출제돼요.

📊 점수 기록

  • 퀴즈 점수가 기록돼요.
  • 자주 틀리는 단어는 다시 나와요.
  • 점수가 올라가는 것을 확인해 보세요!
💡 어휘퀴즈를 자주 풀면 어휘력이 쑥쑥 늘어요!
14 🏪 CU 편의점 응모 ▼

🎁 CU 편의점 응모란?

  • 학습을 열심히 하면 응모권을 받을 수 있어요.
  • 응모권으로 CU 상품권 추첨에 참여해요!

🎯 응모권 받는 방법

  • 단원 학습 완료 시 응모권 획득
  • 관문 통과 시 추가 응모권
  • 연속 학습 보너스 응모권

📋 응모 방법

  • 응모권이 모이면 [응모하기] 버튼 클릭
  • 당첨 결과는 매월 발표돼요!
💡 열심히 공부하면 상품도 받을 수 있어요! 🎉
15 📢 확성기 (공지사항) ▼

📣 확성기란?

  • 선생님이나 관리자가 보내는 공지사항이에요.
  • 중요한 소식을 확인할 수 있어요.

📌 확인할 수 있는 내용

  • 이벤트 알림: 특별 이벤트 안내
  • 공지사항: 시스템 업데이트, 점검 안내
  • 학습 팁: 효과적인 학습 방법 안내
  • 축하 메시지: 특별한 성취 축하
ℹ️ 확성기 알림이 오면 꼭 확인해 보세요!
16 🏆 월간 스피드 어휘왕 ▼

🎯 스피드 어휘왕이란?

  • 한 달 동안 어휘 퀴즈 점수를 겨루는 대회예요.
  • 전국의 친구들과 순위를 경쟁해요!

📊 순위 결정 방식

  • 어휘 퀴즈 정답률
  • 풀이 속도
  • 총 푼 문제 수

🎁 보상

  • 1등: 특별 배지 + 상품
  • 2~10등: 스페셜 배지
  • 참여상: 참여자 전원 배지
💡 매월 1일에 순위가 초기화돼요. 도전해 보세요! 🚀
17 📚 월간 스피드 독서왕 ▼

📖 스피드 독서왕이란?

  • 한 달 동안 완독한 단원 수를 겨루는 대회예요.
  • 누가 가장 많이 읽었는지 순위를 확인해요!

📊 순위 결정 방식

  • 완독한 단원 수
  • 총 학습 시간
  • 학습 완료율

🎁 보상

  • 1등: 독서왕 트로피 배지 + 상품
  • 2~10등: 골드 독서 배지
  • 참여상: 독서 인증 배지
⚠️ 대충 넘기지 말고 꼼꼼히 읽어야 점수가 올라가요!
18 📈 주간분석 ▼

📊 주간분석이란?

  • 일주일 동안의 학습 통계를 보여줘요.
  • 화면 오른쪽 위에서 확인할 수 있어요.

📋 확인할 수 있는 내용

  • 완료한 단원 수: 이번 주에 끝낸 단원
  • 영역별 평균 점수: 5가지 영역 점수
  • 학습 시간: 일별/주간 학습 시간
  • 성장 추이: 지난주 대비 변화
ℹ️ 매주 월요일에 새로운 주간 분석이 시작돼요!
