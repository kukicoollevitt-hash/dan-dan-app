관문 클릭 시

해당 학생의 세부 관문 문해 행동 데이터를 확인할 수 있습니다.

🔄
재도전 횟수 / 총 시도 횟수

학생이 관문을 몇 번 재시도했는지, 총 몇 번 도전했는지 확인합니다.

⏱️
누적 전체 풀이 시간

관문을 통과하기까지 총 소요된 시간을 확인합니다.

❌
누적 전체 오답 클릭

관문 풀이 중 총 몇 번 오답을 선택했는지 확인합니다.

⚠️
가장 미흡한 문항

오답이 많거나 시간이 오래 걸린 문항을 상위 2개로 표시하여 보완이 필요한 영역을 파악합니다.

📊
각 문항별 지수 / 누적 시간 / 누적 오답

10개 문항 각각의 상세 데이터를 확인하여 학생의 취약점을 분석합니다.

📖 본문학습 행동 데이터
⏰
평균 독해시간 / 단원 총 소요시간

문해 소요시간을 통해 과목별 평균보다 보완이 필요한 단원을 체크할 수 있습니다. 너무 빠르거나 너무 느린 경우 주의가 필요합니다.

📝
중심문단 총 오답

해당 단원의 문단별 오답 횟수를 체크하여 실제로 보완이 필요한 단원을 확인할 수 있습니다. 클릭하면 상세 내역을 볼 수 있습니다.

🎯
문제 총 오답

지수별 문제별 오답 횟수를 체크하여 실제로 보완이 필요한 단원을 확인할 수 있습니다. 클릭하면 상세 내역을 볼 수 있습니다.

💡
활용 팁

오답이 많은 학생은 해당 단원의 보완학습을 권장하고, 독해시간이 평균보다 현저히 짧은 경우 꼼꼼히 읽기를 지도해 주세요.

📝 진단테스트 메뉴 가이드

📋
진단테스트 신청 목록

최초 신청한 상담 학생의 학생 및 학부모 정보를 수집하고, 상담 시 활용할 수 있습니다.

📊
진단테스트 결과 목록

진단테스트 모든 문항 완료 시 자동으로 학생 시리즈(레벨) 및 시험지가 등록됩니다.

📄
시험지 보기 (적극 활용!)

해당 학생의 진단테스트 시험지를 확인할 수 있습니다. 상담 시 필수 활용 자료입니다.

📑 시험지 내용 및 요소
✅
기본 정보

정답 및 오답, 응시일, 총 시험 소요시간을 확인할 수 있습니다.

📈
5가지 문해력 지수 레이더

사실적/구조적/어휘력/추론적/비판적 이해력을 레이더 차트로 시각화합니다.

🎯
추천 시리즈

학생 수준을 고려한 시리즈 추천입니다. 참고용으로 활용해 주세요.

💪
강점 및 성장 영역

학생의 강점과 앞으로 성장이 필요한 영역을 파악할 수 있습니다.

📊
문해력 지수별 정답률 (상담 필수!)

각 문해력 지수별 정답률을 상담 시 적극 활용하세요.

📚
과목별 정답률 (상담 필수!)

과목별 정답률도 상담 시 적극 활용하세요.

🤖
AI 맞춤 분석 피드백

시험지를 자동 분석하여 피드백이 반영됩니다. 미흡할 시 새로고침해 주세요.

🔍
틀린 문항 바로가기

상담 시 틀린 문항을 바로 확인하여 설명할 수 있습니다.

💡
문제별 정답 및 해설

해당 문제의 정답과 해설을 상담 시 활용할 수 있습니다.

📚 학생 관리 메뉴 가이드

➕
학생 추가

신규 학생을 간편하게 추가할 수 있습니다. 학생 정보(이름, 학년, 연락처 등)를 입력하여 등록합니다.

🔍
이름 검색

검색창에 학생 이름을 입력하면 해당 학생을 빠르게 찾을 수 있습니다.

👤
학생 이름 클릭

테이블의 학생 이름을 클릭하면 해당 학생의 화면으로 바로 이동합니다. 학습실, 종합리포트 등 전반적인 정보를 확인하고 수정할 수 있습니다.

📊
학습실 상태

해당 학생의 전체 과제 및 완료된 과제 현황을 확인할 수 있습니다. 클릭하면 학생 학습실로 바로 이동합니다.

✅
상태 (승인/대기 전환)

신규 학생이 회원가입하면 기본적으로 '대기' 상태입니다. 선생님이 확인 후 '승인'으로 전환하거나, 필요시 다시 '대기'로 변경할 수 있습니다.

🎯
시리즈 부여

진단테스트 결과를 고려하여 학생에게 적합한 시리즈(레벨)를 부여합니다. 브레인온/업/핏/딥 중 선택 가능합니다.

📱
과제 알림

알림 버튼을 클릭하면 카카오톡으로 해당 학생(또는 학부모)에게 과제 알림 메시지를 발송할 수 있습니다.

✏️
수정

수정 버튼을 클릭하면 학생의 정보(이름, 학년, 연락처, 소속반 등)를 수정할 수 있습니다.

🗑️
삭제

삭제 버튼을 클릭하면 해당 학생의 모든 데이터가 삭제됩니다. 학습 기록, 리포트 등 모든 정보가 복구 불가능하게 삭제되므로 신중하게 사용해 주세요.

⚠️
전체 데이터 삭제

모든 학생의 데이터가 일괄 삭제됩니다. 학원 전체 학생의 학습 기록이 복구 불가능하게 삭제되므로 절대 주의가 필요합니다!
