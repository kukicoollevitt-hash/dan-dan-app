제1장파동이란

🌊 에너지를 전달하는 물결

연못에 돌을 던지면 물결이 퍼져나갑니다. 하지만 물 자체가 이동하는 건 아닙니다. 이것이 바로 파동입니다.

📖 파동(Wave)의 정의:
• 진동이 매질을 따라 퍼져나가는 현상
• 에너지는 전달되지만, 매질은 이동하지 않음
• 매질: 파동을 전달하는 물질 (공기, 물, 고체 등)

파동의 예:
• 수면파, 음파, 지진파, 줄의 파동
• 빛(전자기파)은 매질 없이도 전달!

파도 위의 배는 위아래로 출렁일 뿐, 파도와 함께 이동하지 않습니다.

제2장파동의 요소

📏 파동을 설명하는 용어들

🔬 파동의 기본 요소:
• 진폭(A): 평형 위치에서 최대 변위까지의 거리
• 파장(λ): 한 파동의 길이 (마루~마루 또는 골~골)
• 주기(T): 한 번 진동하는 데 걸리는 시간 (초)
• 진동수(f): 1초 동안의 진동 횟수 (Hz)
• 속력(v): 파동이 전파되는 속도 (m/s)

관계식: f = 1/T, v = fλ = λ/T

마루: 파동의 가장 높은 점
골: 파동의 가장 낮은 점

진폭이 클수록 에너지가 크고, 진동수가 높을수록 빠르게 진동합니다.

제3장파동 공식

🧮 v = fλ 마스터하기

📖 파동의 기본 공식:

v = fλ = λ/T

• v: 파동 속력 (m/s)
• f: 진동수 (Hz)
• λ: 파장 (m)
• T: 주기 (s)

변형 공식:
• λ = v/f = vT
• f = v/λ

예제: 진동수 500Hz, 파장 0.68m인 음파의 속력은?

v = fλ = 500 × 0.68 = 340 m/s

예제2: 속력 340m/s, 진동수 170Hz인 파동의 파장은?

λ = v/f = 340/170 = 2 m

제4장횡파와 종파

↔️ 두 가지 파동의 형태

📖 횡파(Transverse Wave):
• 진동 방향 ⊥ 파동 진행 방향
• 마루와 골이 번갈아 나타남
• 예: 빛, 줄의 파동, 지진의 S파

종파(Longitudinal Wave):
• 진동 방향 ∥ 파동 진행 방향
• 밀(압축)과 소(팽창)가 번갈아 나타남
• 예: 소리, 용수철 파동, 지진의 P파

소리는 대표적인 종파입니다. 공기 분자가 앞뒤로 진동하면서 압축과 팽창이 전달됩니다.

빛은 횡파입니다. 전기장과 자기장이 진행 방향에 수직으로 진동합니다.

제5장중첩 원리

➕ 파동이 만나면 어떻게 될까?

🔬 중첩 원리(Superposition Principle):
• 두 파동이 만나면 각 점의 변위가 합쳐짐
• 만난 후에는 서로 영향 없이 원래대로 진행

보강 간섭 (Constructive):
• 마루 + 마루 = 더 큰 마루
• 골 + 골 = 더 깊은 골
• 진폭 증가

상쇄 간섭 (Destructive):
• 마루 + 골 = 상쇄
• 진폭 감소 또는 0

노이즈 캔슬링 이어폰의 원리: 외부 소음과 상쇄 간섭하는 파동을 발생시켜 소음을 없앱니다!

제6장간섭

🎯 두 파원의 간섭 무늬

📖 두 파원의 간섭:
• 같은 진동수의 두 파원이 파동을 발생
• 공간에 보강/상쇄 지점이 교대로 나타남

보강 간섭 조건:
경로차 = nλ (n = 0, 1, 2, ...)

상쇄 간섭 조건:
경로차 = (n + ½)λ (n = 0, 1, 2, ...)

※ 경로차: 두 파원에서 관측점까지의 거리 차이

물탱크에서 두 점이 동시에 진동하면, 수면에 간섭 무늬가 나타납니다.

밝은 띠(보강)와 어두운 띠(상쇄)가 번갈아 나타나요!

제7장반사

🔄 파동의 반사

📖 파동의 반사 법칙:
• 입사각 = 반사각 (법선 기준)
• 파장, 진동수, 속력은 변하지 않음

고정단 반사:
• 끝이 고정된 경우
• 반사파가 위상 반전 (마루↔골)

자유단 반사:
• 끝이 자유로운 경우
• 반사파가 위상 유지 (마루→마루)

줄의 한쪽 끝이 벽에 고정되어 있으면: 마루가 반사되어 골로 돌아옵니다.

줄의 한쪽 끝이 자유롭게 움직이면: 마루가 반사되어 마루로 돌아옵니다.

제8장정상파

🎸 제자리에서 진동하는 파동

🔬 정상파(Standing Wave):
• 진행하지 않고 제자리에서 진동하는 파동
• 진행파와 반사파의 중첩으로 형성

정상파의 특징:
• 배(Antinode): 최대 진폭으로 진동하는 점
• 마디(Node): 진동하지 않는 점 (변위 항상 0)
• 배와 마디 사이 거리 = λ/4
• 인접한 마디 사이 거리 = λ/2

기타 줄을 튕기면 정상파가 생깁니다. 양 끝이 고정되어 있으므로 특정 파장만 가능해요!

"정상파 = 진행파 + 반사파의 합작품!"

제9장공명

🔔 진동이 증폭되는 순간

📖 공명(Resonance):
• 외부 진동수가 물체의 고유 진동수와 일치할 때
• 진폭이 급격히 증가하는 현상

고유 진동수:
• 물체가 자연스럽게 진동하는 진동수
• 크기, 모양, 재질에 따라 결정

공명의 예:
• 그네를 타이밍 맞춰 밀기
• 와인잔을 목소리로 깨기
• 다리의 공진 (타코마 다리 붕괴)

공명은 에너지가 효율적으로 전달되어 진폭이 커지는 현상입니다.

군인들이 다리를 건널 때 발을 맞추지 않는 이유: 공명으로 다리가 무너질 수 있어서!

제10장파동의 응용

📡 일상 속 파동 기술

📖 파동의 실생활 응용:
• 라디오/TV: 전자기파로 정보 전송
• 초음파 검사: 반사파로 내부 영상 생성
• 지진계: 지진파 분석으로 진원 위치 파악
• 악기: 정상파와 공명으로 소리 발생
• 전자레인지: 마이크로파로 물 분자 진동
• 와이파이: 전자기파로 무선 통신

휴대폰 통화가 가능한 이유: 전자기파가 기지국과 정보를 주고받기 때문!

의료 초음파: 초음파가 조직에서 반사되는 패턴을 분석해 영상을 만듭니다.

"보이지 않는 파동이 현대 문명을 움직인다!"
