제1장굴절이란

🌊 왜 빛은 꺾일까?

수영장에 발을 담그면 다리가 짧아 보입니다. 물컵 속 빨대가 꺾여 보이기도 하죠. 이 모든 현상의 원인은 빛의 굴절입니다.

📖 굴절(Refraction)이란:
• 빛이 한 매질에서 다른 매질로 진행할 때 경계면에서 진행 방향이 바뀌는 현상
• 원인: 매질마다 빛의 속력이 다르기 때문
• 빠른 매질 → 느린 매질: 법선 쪽으로 꺾임
• 느린 매질 → 빠른 매질: 법선에서 멀어지게 꺾임

빛의 속력: 진공 > 공기 > 물 > 유리
매질이 밀할수록(굴절률이 클수록) 빛이 느려집니다.

제2장굴절률

📊 굴절률의 의미

🔬 굴절률(Refractive Index, n):

n = c / v
(c: 진공에서 빛의 속력, v: 매질에서 빛의 속력)

• 진공: n = 1.00
• 공기: n ≈ 1.00
• 물: n = 1.33
• 유리: n ≈ 1.5
• 다이아몬드: n = 2.42

굴절률이 클수록 빛이 그 매질에서 느리게 진행합니다.

다이아몬드가 반짝이는 이유? 굴절률이 매우 커서(2.42) 빛이 내부에서 여러 번 반사되기 때문입니다!

"굴절률 = 빛을 얼마나 느리게 하느냐의 척도!"

제3장스넬의 법칙

📐 굴절의 수학적 법칙

📖 스넬의 법칙(Snell's Law):

n₁ sin θ₁ = n₂ sin θ₂

• n₁, n₂: 각 매질의 굴절률
• θ₁: 입사각 (법선과 입사광선 사이 각)
• θ₂: 굴절각 (법선과 굴절광선 사이 각)

또는: n₁/n₂ = sin θ₂/sin θ₁ = v₁/v₂

예제: 공기(n=1)에서 물(n=1.33)로 30° 각도로 입사할 때

1 × sin 30° = 1.33 × sin θ₂
sin θ₂ = 0.5 / 1.33 = 0.376
θ₂ = 22.1°

굴절률이 큰 매질로 들어가면 법선 쪽으로 꺾입니다!

제4장전반사

💎 빛이 100% 반사되는 순간

🔬 전반사(Total Internal Reflection):
• 조건: 굴절률이 큰 매질 → 작은 매질로 진행
• 임계각(θc) 이상의 입사각에서 발생

임계각 공식: sin θc = n₂/n₁
(n₁ > n₂일 때)

• 물→공기: θc ≈ 48.6°
• 유리→공기: θc ≈ 42°
• 다이아몬드→공기: θc ≈ 24.4°

전반사는 빛이 굴절하지 않고 경계면에서 완전히 반사되는 현상입니다.

다이아몬드의 임계각이 작아서(24.4°) 내부에 들어간 빛이 쉽게 빠져나가지 못하고 여러 번 반사됩니다. 그래서 찬란하게 빛나죠!

제5장광섬유

🌐 전반사의 놀라운 응용

📖 광섬유(Optical Fiber):
• 원리: 전반사를 이용한 빛의 전송
• 구조: 코어(굴절률 큼) + 클래딩(굴절률 작음)
• 빛이 코어-클래딩 경계에서 전반사 반복
• 장점: 손실 적음, 대용량 전송, 전자기파 방해 없음

인터넷 데이터가 바다 건너 전송되는 비결? 바로 광섬유 케이블입니다!

광섬유 활용:
• 초고속 인터넷 통신
• 내시경 (의료용)
• 조명 장식
• 센서 시스템

"전반사로 빛을 가둬 세상 끝까지 보낸다!"

제6장볼록렌즈

🔍 빛을 모으는 볼록렌즈

📖 볼록렌즈(Convex Lens):
• 모양: 가운데가 두꺼움
• 기능: 평행광선을 한 점(초점)에 모음
• 별명: 수렴 렌즈
• 초점거리(f): 양수 (+)

렌즈 방정식: 1/f = 1/p + 1/q

물체 위치에 따른 볼록렌즈의 상:

• 2f 바깥: 축소 도립 실상
• 2f 위치: 같은 크기 도립 실상
• f~2f: 확대 도립 실상
• f 안쪽: 확대 정립 허상 (돋보기 원리!)

제7장오목렌즈

👓 빛을 퍼뜨리는 오목렌즈

📖 오목렌즈(Concave Lens):
• 모양: 가운데가 얇음
• 기능: 평행광선을 퍼뜨림 (발산)
• 별명: 발산 렌즈
• 초점거리(f): 음수 (-)
• 상: 항상 축소된 정립 허상

오목렌즈는 항상 작고 바로 선 허상만 만듭니다.

근시 교정에 오목렌즈를 쓰는 이유:
근시안은 상이 망막 앞에 맺히므로, 오목렌즈로 빛을 퍼뜨려 상이 망막에 정확히 맺히도록 합니다.

"오목렌즈 = 축소 + 정립 + 허상, 언제나!"

제8장렌즈 작도

✏️ 렌즈의 광선 작도법

🔬 볼록렌즈 광선 작도 규칙:
1. 축과 평행한 광선 → 굴절 후 초점(F) 통과
2. 렌즈 중심 통과 → 직진 (꺾이지 않음)
3. 초점 통과 광선 → 굴절 후 축과 평행

오목렌즈:
• 평행광선 → 허초점에서 나오는 것처럼 발산

렌즈에서 상 찾기: 물체의 한 점에서 2개 이상의 광선을 그려 교점을 찾습니다.

배율(m) = -q/p = 상의 높이/물체의 높이
• m > 0: 정립 (같은 방향)
• m < 0: 도립 (반대 방향)

제9장분산

🌈 무지개가 생기는 이유

📖 빛의 분산(Dispersion):
• 백색광이 프리즘을 통과하면 여러 색으로 나뉨
• 원인: 색마다 굴절률이 다름
• 보라 > 파랑 > 초록 > 노랑 > 주황 > 빨강 (굴절률 순)
• 빨간빛: 굴절 적게, 보라빛: 굴절 많이

뉴턴은 프리즘 실험으로 백색광이 여러 색의 빛으로 이루어져 있음을 증명했습니다.

무지개는 빗방울이 작은 프리즘 역할을 해서 태양빛을 분산시킨 결과입니다!

색 순서: 빨주노초파남보 (파장이 긴 순서)

제10장광학기기

📷 렌즈의 다양한 응용

📖 광학기기 속 렌즈:
• 카메라: 볼록렌즈로 실상을 필름/센서에 맺음
• 눈: 수정체(볼록렌즈)로 망막에 상 형성
• 현미경: 대물렌즈 + 접안렌즈 (확대)
• 망원경: 대물렌즈 + 접안렌즈 (원거리 관측)
• 안경: 근시-오목, 원시-볼록

우리 눈도 볼록렌즈 시스템입니다! 수정체가 빛을 굴절시켜 망막에 상을 맺게 해요.

• 근시: 상이 망막 앞에 맺힘 → 오목렌즈 교정
• 원시: 상이 망막 뒤에 맺힘 → 볼록렌즈 교정

"굴절의 원리로 세상을 더 크게, 더 멀리 본다!"
